暗流城Shadycreek
人口：14,770（56% 人类，15% 精灵，14% 矮人，15% 其他种族）
政府：四个有组织的犯罪家族和公民经营的灰商联盟争夺城市的控制权，而怨恨帮的雇佣维和部队则试图阻止彻底的混乱。
防御：Shadycreek Run 的帮派们各自雇佣私人士兵和保镖，而怨恨帮则保卫着整个城市。
商业：大多数普通商品都可以买到，稀有进口商品和被盗商品被秘密出售和拍卖。许多小酒馆、旅馆和罪恶场所都向公众开放。
组织：大多数企业由部落或联盟之一经营，即使联盟是隐藏的。这个城市有分散的，简陋的寺庙。经过怀南迪尔的北部前哨，一排分散的棚户区和小屋将小路堵在一条小溪旁，通往荒地的树木。
最初由残酷的贫民窟领主经营的临时贫困社会拼凑而成，Shadycreek Run 现在被称为犯罪活动和野蛮人性的蜂巢，自由统治。堕落的文明渣滓将这里视为获得无数名声和权力的机会。大多数辩论以斗殴告终，而大多数斗殴以流血而告终。
严酷、无人看守的环境孕育了坚韧的民族，尽管有肆虐的野兽从郊区捕食容易的猎物，但自豪的猎人和剥皮者群体已经崛起成为杰出的贸易商。
一些聪明的人物已经开始在 Run 中组织和文明生活，希望为收集Reprobates 带来一点秩序和尊重，而 Run 的退伍军人则努力让事情变得不可预测，以暴力破坏这些尝试。动荡的社会强调了这句古老的格言，“他们称其为跑步是有原因的。”

危险的职业Dangerous Vocations
许多选择住在Shadycreek Run的人这样做是为了避免帝国内部等待的迫害(或起诉)，而另一些人则喜欢在这片公开的道德灵活性的土地上享受残酷的生活方式。这里的职业从工匠、工匠到农民、雇佣兵和艺人都有。然而，该地区最赚钱的企业家是小偷、猎人和拾荒者，他们勇敢地面对致命的萨瓦里尔森林和莫拉斯米尔的废墟。强壮的野兽能产出丰盛的肉和毛皮，而恢复的文物和丢失的魔法在遥远的市场上能卖到相当高的价格。争夺这些战利品的斗争常常变得血腥，部落和灰商人联盟为获得这些贸易的控制权而战斗，同时也与乌托敦的人们竞争，后者希望收回他们的遗产和丢失的物品。

竞争关系Adversarial Rulership
目前，Shadycreek Run分为三个竞争派系：Greytrader联盟、部落Tribes和仇恨帮派Grudge Gang。
格雷特工会（Greytrader Union）是一个由改革后的被驱逐者和罪犯组成的派别，他们希望与外部社会建立牢固的贸易关系，并通过使Shadycreek成为一个更安全、功能更强大的社区来帮助挽救Shadycreek。工会成员不遵守帝国统治，他们享受着无法无天的生活方式的自由，但他们也希望摆脱Shadycreek Run众所周知的暴力遗产。由各行各业的捕猎者、猎人、建设者和工匠组成的格雷特工会希望成立自封的元帅来维持和平。他们的目标是增加平民的安全，将逃亡变成一个体面的贸易目的地，并最终减少部落和仇恨团伙的影响。
这些部落是敌对的犯罪家族的集合，他们在暴力内讧和内部权力斗争中失去了大部分统治地位之前，已经逃亡了近一个世纪。这些部落由马尔登、乌托洛特、特雷班和贾根托斯家族组成，致力于控制城市中发生的大部分业务，并秘密相互勾结。虽然每个家庭中心通常由血缘或婚姻关系的亲属组成，但只要他们保持忠诚，他们的员工也被视为“家庭内部”。部落围堵赃物，提供邪恶和淫秽的娱乐，并通过混乱、恐吓和恐惧控制民众。
怀恨在心的帮派是一群顽固的雇佣军，他们充当力量，在逃亡期间强制执行当地利益。由德文达利军队的一队逃兵建立，他们已经成长为一个忠于硬币和彼此的战士营。只要他们的生意不至于血腥到伤害无辜者或危及逃跑的安全，帮派成员将很乐意为格雷特工会或部落的任何家庭工作。因为其他派系依靠仇恨帮派来保护自己免受
在萨瓦里之森Savalirwood，其成员认为自己是政治棋局中的中立工具。该团伙的一些成员指派自己在一片没有法律的土地上维持秩序，称自己为任务者，并尽最大努力防止无辜者被城市的掠夺者压榨。他们的首要任务是把他们的技能借给支付得最好的人，维护宜居的街道，防止萨瓦里之森和敦达里安帝国的危险进入城市。
政府Government
这座城市没有中央政府，也没有法律，所以它在很大程度上是由一种模糊的“生活和让别人生活”的观念维系在一起的，偶尔也会被“拿我想要的”的情绪打破。如果暴力活动失控，咒怨帮的任务员就会介入，把所有参与者打成肉酱。粗心大意的人死在被雪覆盖的小巷里，身上的东西被剥光，被饥肠辘辘的狗吃掉，这并不罕见。逃亡部落的居民学会了提防他们的背部和小心地挥舞刀剑。
地理位置Geography
最初是沿着萨瓦里森林南部的林木线建造的一长串棚户区和棚屋，现在已经变成了一条三英里长、弯曲的主街，被称为三叶草巷，两旁是小屋、建筑物和泥坑。小巷南侧的社区被称为“南三叶草”（South Clover），是这座城市最贫困、维护最少的地区，以极度绝望的居民和更危险的生意而闻名。北部被称为北三叶草，部分建在萨瓦里尔森林中，居住着富裕而有权势的居民，包括大多数部落的家庭，他们住在各种豪宅和小堡垒里。
三叶草巷的西端被称为Wickerskids，是食腐动物和农民的家园，他们竭尽全力在寒冷的北方贫瘠土地上为暗流邦Shadycreek Run的居民生产食物。三叶草巷的中心广场被命名为三叶草广场，标志着城市商业、冲突和罪恶的中心。在广场上，所有派系在光天化日之下混杂在一起，用一种放纵和过度的淫秽和喧闹的生活方式来掩盖他们隐藏的议程。四叶草巷的东端被称为格拉夫贫民窟，住着一群农民和扒手，他们住在毒品窝点和猪圈里。

暗影的冒险Shadycreek Run Adventures
宽松的法律和轻松赚钱的承诺将许多机会主义者吸引到这些危险的街道上。
一颗被盗的心（低级）A Stolen Heart (Low Level)。在 Trench Tavern，角色们加入了与当地怨恨帮的饮酒游戏。无论输赢，该团伙的一名矿石成员 Dratto 透露了他们与 Mardoon 家族的联系。 Dratto 提到他们需要一些人来窃取目前由 Trebain 家族持有的 Molaesmyr 遗物。角色可以将 Dratto 变成 Trebains 以获得他们的青睐或同意协助盗窃。
如果角色同意帮助 Dratto，就会发生危险的入室盗窃。他们必须潜入 Wickerskids 外面的一个守卫棚屋，才发现这件遗物实际上是一个沉睡的小精灵。这个生物在角色面前醒来，当一个角色天性的一个神秘方面将小精灵从它的麻木中拉出来并将它的感情与角色联系在一起时。
国王的间谍（任何级别）King's Spy (Any Level)。 Nella Durlon（守序中立，非女性，人类间谍）作为帝国间谍居住在 Shadycreek Run。他们收集有关该定居点防御的信息并带回给 Oliver Schrieber（参见第 2 章中的“Dwendalian 帝国”）。角色们从雇佣他们寻找间谍的派系中得知了 Nelia 的工作。根据阵营的不同，角色必须要么护送内拉安全离开定居点，以便向奥利弗报告，要么要求杀死内拉以泄露定居点的秘密。无论哪种方式，找到内拉都不是一件容易的事。间谍精通他们的手艺，并与逃跑的罪犯和恶棍混在一起。
