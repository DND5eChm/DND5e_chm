相互尊重 Mutual Respect

无论你是在给亲友还是给陌生人开团，奠定彼此间的信任都十分重要。要知道，最好的团是要在一桌人都能有足够的安全感来展现个性、畅所欲言、代入角色时才能跑出来的。
所有人都应当坚守“尊重”这一原则。如果有人违反了这一点，那通常就得由DM出面解决。但这并不绝对――如果一个玩家的行为影响到了其他所有人的兴致，那所有人都该帮DM解决这一问题。

建立预期 Setting Expectations

在为你的桌找人之前，你需要为你想拉过来的玩家描述你想开的团――写出这个虚幻世界中可能发生的冲突、这个设定总体基调、还有你想探寻的游戏主题。（本章前述的“每个DM都是独一无二的Every Dm Is Unique”章节能助你向他人描述自己的团。）
当玩家们想象自己能车什么样的角色时，你可以告诉玩家他们能期待什么样的内容，并询问玩家们，他们对游戏中可能出现的哪些要素的可以接受的，哪些又是应当避免的。讨论这些内容不需要透露你故事中的剧情与转折，但你应当告知你想在团内探讨的主题、分享你受到了哪些故事的启发、还有你对哪种奇幻风格更感兴趣（在第五章中对此有概述）――你得在开团前让玩家明白这是不是他们想跑的团。
明确你自己的游戏预期，同时确保自己理解玩家的游戏预期，可以帮助你更流畅地主持游戏。慎重对待你的玩家的意见和期待，并让玩家在对待你的意见和期待时同样慎重。理想情况下，你会找到一种适合所有人的游玩风格。

硬底线与软底线 Hard And Soft Limits

除了战役的主题与奇幻风格，询问玩家对方有哪些觉得敏感或不舒服的内容也很重要。进行此类讨论时可以划分软硬两种底线：

  • 软底线代表那些应慎重使用的内容，它们可能会导致无法接受的焦虑、恐惧或难受。
  • 硬底线则是那些根本不该提及或描述的内容。

DM和玩家都可能会有其他人没能意识到的恐惧症或诱因。任何让大家感到不安全感（硬底线）的游戏内主题和内容都应该避免。如果某个主题或内容让一个或多个玩家感到焦虑，但是他们同意将其加入游戏中（弱底线），请保持慎重，并做好随时将这些内容移除的准备。
通常来说，游戏内的底线用于例如角色间的恋爱、性爱、压迫操控、种族主义、奴役以及对未成年人和动物的暴力之类的内容。底线也可以用于具体的生物，例如蜘蛛、蛇、老鼠和恶魔。谈论“什么样的灾厄可能会降临到角色的头上”这样的底线同样十分重要，例如精神控制类魔法、无助感、还有死亡。
尽管上边说了那么多，D&D还是一个包含有虚拟冲突和混乱的游戏。游戏中必然具有的核心元素是难以抛掉的。例如，受到伤害并不是一个能轻易解决的底线。类似的，角色死亡有时也是无法阻止的事，不过在游戏中仍有许多办法去消除或避免它（见第三章“死亡Death”一节获取建议）。

使用游戏预期卡 Using The Game Expectations Sheet

游戏预期跟进卡是用来在游戏开始时确立预期，以确保每个人都能玩得开心的工具。
在给玩家们发卡之前，请你填写顶部的这两栏：

  • 游戏主题与风格 Game Theme and Flavor：在本栏，大致描述一下你对游戏内容设想的方向。见“建立预期Setting Expectations”一节获取其中包含的各种信息。
  • 潜在敏感元素 Potentially Sensitive Elements：如果你知道游戏中的一些元素可能会触碰到玩家的底线，就在本栏中将它们列出来。见“硬底线与软底线Hard and Soft Limits”一节获取例子。

填好上述信息后请把卡的复印件发给每位玩家。玩家填卡时可以选择匿名，但以下信息不该省略：

  • 底线 Limits：使用叉号作为硬底线并用问号作为软底线，以此标明哪些元素是可能会导致问题的潜在敏感元素、又有哪些元素根本不应加入。
  • 希望，预期和担忧 Hopes,Expectations,and Concerns：填写者需要在最后两个栏里分享自己对游戏任何的希望和预期，并且列出对本团表现的担忧。

收集全部的卡，并将玩家们的底线汇总成一份全团人都可查看的独立匿名文档。


交流底线 Communicating Limits：讨论底线时请确保你所用的方式不会让对方感到不适。玩家们可能并不愿意公开讨论自己的底线，尤其对于那些角色扮演游戏的新手，以及跟团里其他人并不熟悉的玩家而言。匿名分享底线可以缓解这种不适感――通过匿名调查来收集玩家们的底线，例如使用本章中的游戏预期跟进卡。
将底线汇总成一份团内共享的列表。底线不容商议，团内的所有人都要尊重这些底线。
这类问题最好在战役开始时讨论，但如果有新玩家入队，或是战役的故事或风格发生转变时，亦需要进行进一步的讨论。有些人可能无意间触及到了某个底线，此时需要进行提醒；另一些人则可能是在游玩过程中才发觉某些内容是自己的底线――即使战役开始前早就讨论过这类内容。同样的，随着战役的展开，玩家们说不定也会发觉新的底线。DM最好每隔几回团就确认一下：有没有人感觉自己可能对游戏的发展感到不适？如果有，就按需更新团内底线。
改变底线 Shifting Limits：不管是私下询问还是当面提出，你都应该鼓励玩家及时补充或提出新的底线，并将其添加到列表之中。请相信玩家们最懂自己的需求，并据此来更新游戏。
游玩中的底线 Limits in Play：D&D是即兴的，所以游戏中可能会出现出乎意料的走向。为了避免触及底线造成恶劣影响，玩家们可以使用一种公认的特定标识来表明“当前的内容突破了底线”。这能够让你迅速调整。这个标识可以是某种姿势（例如交叉胳膊形成“X”形，或举起手比出“停止”的姿势），一串密语或词，身体接触或是举起一个专用的物品，又或是其他你团队中所有人认可的方式。其他玩家们也不应质疑有人提出“停止”，并停止游戏，直到问题得以解决。使用停止标识的玩家可以评论他认为需要调整的内容，但无需解释为什么自己对这些内容有异议。停止标识不应引起争议或讨论：请感谢玩家们如实表达自己的需求。此后你需要正确重设场景，继续游戏。
你应当让玩家们清楚知晓：如果有人不愿使用停止标识，那他也可以选择离开游戏，或申请游戏暂停，并与你私下谈谈。玩家还可能会让自己的朋友代为使用标识。作为DM，你要以身作则：严肃对待玩家们的需求，使用一切可用的工具来调整故事的走向。

团内矛盾 Intra-party Conflict

当冒险队伍中的玩家或冒险者之间发生了冲突，通常预示着以下三种情况：

  • 搞破坏的玩家 Disruptive Player：一名玩家在游戏中表现出了反社会行为。“反社会行为 Antisocial Behavior ”一节包含了如何处理这种情况。
  • 玩家冲突 Player Conflict：角色间的冲突有时是玩家间冲突的表象。这些冲突最好在远离游戏桌的地方解决。请鼓励玩家们在游戏之外解决他们之间的冲突。如果桌内的冲突持续升温，你可能需要让他们暂时离开战役，甚至直接离开游戏。
  • 角色扮演 Roleplaying：角色间的冲突并不总是不好的。例如：该怎么处置俘虏的敌人、或是该在即将到来的战争中支持哪方势力――角色（和玩家）对于这样的问题上产生异议是完全正常且可行的。不过如果异议发展得太过火了，请暂停游戏，跳出角色的视角，讨论玩家们想要如何进行。

如果你无法辨识角色或玩家们是因上述哪种情况而冲突，请和他们谈一谈。。
